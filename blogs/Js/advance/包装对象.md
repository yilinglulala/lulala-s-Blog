

```js
var str = "a b c"
str.split(' ')
```

string 在执行方法时做了啥

```js
function mysplit(str, method, arg) {
    let temp = new String(str); // 先生成临时包装对象
    return temp[method](arg);   // 再执行方法
    // 执行完就会销毁包装对象
}
mysplit(str,'split',' ')// 使用方式
```

注意临时包装对象执行完方法就会被销毁

```js
var str = "a b c"
str.name= "luhan"        // 不会报错
str.name // ->undefined  // 因为属性添加在临时包装对象上，但是立马被销毁了，所以是undefined
```
